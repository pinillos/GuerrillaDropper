# GuerrillaDropper

Un juego de scroll infinito horizontal donde controlas a Papá Noel en un trineo volador. Haz click para dejar caer regalos y apunta a los niños para sumar puntos, evita al Grinch y los árboles no afectan la puntuación.

## Cómo jugar
- Haz click para que Papá Noel deje caer un regalo.
- Los regalos caen a tierra y chocan con objetos:
  - Niño: +1 punto
  - Grinch: -1 punto
  - Árbol: 0 puntos
- La partida dura 60 segundos.
- Al finalizar, se muestra la puntuación y si superaste el récord.

## Instalación
1. Coloca los sprites en la carpeta `resources/`:
   - `santa.png`: Sprite de Papá Noel normal.
   - `santa_drop.png`: Sprite de Papá Noel dejando caer regalo.
   - `gift.png`: Sprite del regalo.
   - `tree.png`: Sprite del árbol de Navidad.
   - `child.png`: Sprite del niño.
   - `grinch.png`: Sprite del Grinch.
2. Abre `index.html` en un navegador.
